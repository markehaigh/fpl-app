# FPL Companion

A Fantasy Premier League companion app. Enter your FPL team ID to see
your current squad, points, rank, price-change predictions and
suggested transfers.

## Run locally
npm install
npm run dev

## Deploy
Push to GitHub and import the repo at vercel.com - no configuration
needed. The api/fpl.js serverless function proxies the official FPL
API so live team data works in the browser.
